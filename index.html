Sobre

mí

Desarrollador web front-end junior autodidacta. Todos los días en este campo se aprende algo nuevo e interesante. Me sigue maravillando la web, cómo una buena fuente y una elección de color o unas pocas líneas de código se vuelven algo muy creativo y genial. Creo que solo un sitio web funcional no es suficiente: debe atraer visualmente y vivir armoniosamente con el contenido que se ha presentado.

Diseño Web

El sitio web estéticamente agradable es más que solo información pixelada, ¿verdad? En mis manos, cada proyecto merece hacer un esfuerzo adicional para encontrar el esquema de color, el diseño y la fuente correctos.
